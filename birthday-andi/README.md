# Happy Birthday Andi - Infinite Celebration Universe

A complete static birthday experience built with HTML, CSS, and JavaScript.

## Run

Open `index.html` in a browser, or serve the folder with any simple static server.

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Features

- Mobile-first nostalgic birthday journey with full-screen scroll stops
- Royal warm palette, readable wishes, and nature imagery
- Lightweight vanilla JavaScript only
- Optional synthesized music-box style birthday audio
- Small click celebration without continuous confetti or canvas effects
- Responsive layout and reduced-motion support

No build tools, frameworks, backend, or npm installation are required.
